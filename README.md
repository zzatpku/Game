# 未名湖守护鸭 Demo

这是根据 `game.txt` 做的迷你可玩演示版。玩家控制一只生活在未名湖的鸭子，处理游客投放到湖里的垃圾，并通过鸣叫或告示牌提醒游客。

## 运行

直接用浏览器打开 `index.html` 即可。也可以在当前目录运行：

```sh
python3 -m http.server 5173
```

然后访问 `http://localhost:5173`。

## 操作

- 移动：`WASD`、方向键，或用鼠标点击湖面目标位置。
- 行动：`Space` 或右侧“行动”按钮。
- 靠近垃圾时行动会叼起垃圾；叼着垃圾靠近左上角回收点时行动会投放。
- 靠近游客时行动会提醒游客。清理 4 件垃圾后会解锁“护湖”告示牌，提醒范围和效果更强。

## 演示重点

- 岸边游客会周期性向湖面投放垃圾。
- 最近垃圾会在湖面高亮，并显示坐标位置。
- 垃圾越多，湖水清澈度下降越快。
- 清理垃圾和提醒游客会改变 HUD 指标，形成可展示的玩法闭环。
